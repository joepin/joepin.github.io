cloudMe

Description

cloudMe is my capstone project that I worked on while studying at General Assembly, and feeds one the passions that drove me to be a developer: the desire to connect everything together in a platform-agnostic manner. The project entails two parts: the first is a server application that a user would download to their device, which enables them to spin up a static web server that serves media files off of their device; and the second is a client application that contains references to the addresses of these web servers so that the user can access their content remotely. For those familiar, my project serves the same purpose that ownCloud does.

ColorFy

Description

ColorFy is an app that I built while at General Assembly, and was borne out of the desire to express the emotions of a piece of text visually. This app introduced me to the world of machine learning (for which I used an API put forth by IBM Watson), and piqued my interest in the filed of color psychology. It was designed to fuse the worlds of psychology, emotion, and computation together, and yields a pretty cool product that end users can enjoy simply. It was this project that taught me to love NoSQL databases, especially MongoDB.
